## 开始使用

1. 安装

使用本框架前，请在CSS中开启 border-box

```
*{box-sizing: border-box;}
*::before{box-sizing: border-box;}
*::after{box-sizing: border-box;}
```

IE 8 及以上浏览器都支持此样式。
